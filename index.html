简要说明 / Quick Summary: AI Nexus 不要求你注册账户。应用当前版本默认不集成广告 SDK，也不依赖第三方统计 SDK。 你在应用中输入的内容会按你选择的功能用途处理，例如本地存储、调用你配置的 AI 接口、或在支持设备上进行可选同步。

1. 适用范围 / Scope

本隐私政策适用于 AI Nexus 应用及与其对应的 GitHub Pages 隐私政策页面。 This Privacy Policy applies to the AI Nexus app and the related GitHub Pages privacy page.

如果你通过 AI Nexus 连接第三方模型服务，第三方服务对数据的处理也将受其自身隐私政策和服务条款约束。 If you connect AI Nexus to a third-party model provider, that provider's privacy policy and terms will also apply.

2. 我们收集和处理的信息 / Information We Collect and Process

2.1 你主动提供的信息 / Information You Provide

  • 聊天内容、提示词、你输入的用户信息和个性化内容。 Conversations, prompts, user info, and personalization content you enter.
  • 你配置的接口地址、模型名称、API Key 以及相关模型参数。 Endpoint URLs, model names, API keys, and related model settings you configure.
  • 你填写的资料信息，例如昵称和邮箱。 Profile information such as nickname and email, if you choose to enter them.
  • 你主动选择的图片或文件附件。 Images or files you explicitly choose to attach.

2.2 应用在本地生成或保存的信息 / Data Generated or Stored Locally

  • 聊天会话、消息时间、模型切换记录和会话标题。 Chat histories, timestamps, model selections, and conversation titles.
  • Token 用量、费用估算、时间范围统计等使用记录。 Token usage, estimated cost, and usage analytics records.
  • 界面主题、语言、偏好设置等设备端配置。 Device-side preferences such as theme, language, and UI settings.

2.3 敏感配置 / Sensitive Configuration

已保存的模型配置和 API Key 会使用系统安全存储机制保存。 In the current app build, saved model profiles and API keys are stored using the system secure storage mechanism.

3. 我们如何使用这些信息 / How We Use Information

  • 提供聊天、模型切换、消息生成和多模型访问能力。 To provide chat, model switching, message generation, and multi-model access.
  • 保存你的会话历史、偏好设置和个人化上下文。 To save your conversation history, preferences, and personalization context.
  • 记录 Token 用量和估算费用，帮助你查看使用趋势。 To record token usage and estimated cost so you can review usage trends.
  • 在你主动触发时，执行模型发现、接口测试和同步操作。 To perform model discovery, connection tests, and sync actions when you explicitly trigger them.

4. 第三方模型服务与网络请求 / Third-Party Model Providers and Network Requests

AI Nexus 会将请求发送到你自己配置的模型接口，例如 OpenAI、Anthropic、Google 或其他兼容服务。 AI Nexus sends requests to the model endpoints you configure, such as OpenAI, Anthropic, Google, or other compatible providers.

  • 当你发送消息时，聊天内容会被发送到你选择的模型服务，用于生成回复。 When you send a message, the conversation content is sent to the provider you selected to generate a response.
  • 当你进行接口测试或模型发现时，应用会向你配置的接口发起网络请求。 When you run connection tests or model discovery, the app sends network requests to the endpoint you configured.
  • 如果你附加图片，且当前模型支持视觉输入，图片数据可能会随请求一起发送。 If you attach an image and the selected model supports vision input, the image data may be transmitted with the request.
  • 当前版本下，非图片文件不会被自动解析并上传给模型；通常只会以文件标识文本加入消息上下文。 In the current version, non-image files are not automatically parsed and uploaded to the model; typically only a file label is inserted into the message context.

这些第三方服务对数据的处理不受 AI Nexus 单独控制。 Their downstream processing is governed by their own policies, not solely by AI Nexus.

5. 数据存储、同步与保留 / Storage, Sync, and Retention

5.1 本地存储 / Local Storage

聊天记录、统计记录、个性化内容、资料信息和大部分配置默认保存在你的设备本地。 Conversations, usage records, personalization content, profile data, and most settings are stored locally on your device by default.

5.2 安全存储 / Secure Storage

已保存的 API Key 和模型配置会通过系统安全存储能力保存。 Saved API keys and model profiles are stored using the operating system's secure storage facilities.

5.3 可选同步 / Optional Sync

在支持的 iOS 设备上，如果你主动开启自动备份或手动执行同步，应用会尝试通过 iCloud Key-Value Store 同步聊天、配置、统计记录、资料信息和共享记忆。 On supported iOS devices, if you enable auto backup or manually trigger sync, the app may sync conversations, configuration, usage records, profile data, and shared memory through iCloud Key-Value Store.

5.4 数据保留 / Retention

数据会保留到你删除相关聊天、清空统计记录、移除安全凭证，或卸载应用为止。 Depending on the operating system, some secure items stored in Keychain may persist across reinstall until you delete them inside the app.

6. 安全措施 / Security

  • 应用使用系统安全存储机制保存敏感模型配置。 The app uses system secure storage for sensitive model configuration.
  • 应用支持系统认证能力，用于解锁安全凭证库等敏感区域。 The app supports system authentication to protect sensitive areas such as the secure vault.
  • 尽管我们采取合理措施保护信息，但任何电子存储或网络传输都无法保证绝对安全。 No electronic storage or transmission method can be guaranteed to be 100% secure.

7. 你的控制权 / Your Controls

  • 你可以修改或删除聊天内容。 You can edit or delete your conversations.
  • 你可以清空 Token 统计记录。 You can clear token usage records.
  • 你可以删除已保存的 API Key 或模型配置。 You can remove saved API keys and model profiles.
  • 你可以关闭自动同步，或不使用 iCloud 同步功能。 You can disable auto sync or avoid using iCloud sync entirely.

8. 儿童隐私 / Children's Privacy

AI Nexus 并非面向 13 岁以下儿童设计。我们不会故意收集儿童个人信息。 AI Nexus is not directed to children under 13, and we do not knowingly collect personal information from children.

9. 政策变更 / Changes to This Policy

我们可能会在功能或数据处理方式发生变化时更新本政策。更新后的版本会发布在本页面，并标注最新更新时间。 We may update this policy when features or data practices change. The latest version will be posted on this page with an updated effective date.
