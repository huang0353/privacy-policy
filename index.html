4-Minute Triage

先做这 4 步，很多问题可以更快定位

01

检查你正在使用的模型配置

确认端点地址、模型名、格式偏好和 API Key 是否对应同一家服务，不要把不同供应商的配置混用。

02

在应用内跑一次连接测试

如果连接测试已经失败，优先排查网络、Key 权限、服务商配额和端点路径，而不是先怀疑聊天页。

03

确认当前模型是否支持你要的能力

图片输入需要模型支持视觉能力。非图片文件在当前版本通常不会被自动解析上传，只会以文件标识加入上下文。

04

区分应用问题和供应商问题

如果接口测试成功但回复异常，可能是模型服务本身的限额、策略或上游波动；如果应用内状态错乱，再联系支持更高效。

Popular Fixes

最常见的 6 类问题

连接失败或没有回复

优先检查接口地址、Key 权限、模型名是否可用，以及服务商账户是否还有额度。

图片传不上去

确认当前模型支持视觉输入，并检查你选择的是图片而不是普通文件。

非图片文件没有被模型读取

当前版本通常不会自动解析并上传非图片文件；它更多是作为文件标识加入消息上下文。

同步看起来没生效

iCloud 同步属于可选功能，只在支持的 iOS 设备和你主动开启时尝试执行。

找不到已保存的 API Key

已保存配置会通过系统安全存储机制保护。先检查是否切换了配置、删除了凭证，或重新录入后验证。

隐私或数据边界不清楚

本页下方已经包含完整隐私政策，可直接查看数据存储、同步、年龄范围和地区适用说明。

FAQ

常见问题

这个页面适用于 iOS 和 macOS 吗？

是的。本页面同时适用于 AI Nexus 的 iOS 和 macOS 版本，可同时作为技术支持页面和隐私政策页面使用。

AI Nexus 面向哪些年龄段？

AI Nexus 面向全年龄段用户。若未成年人使用，建议在父母或监护人指导下配置模型服务、API Key 和网络访问。

AI Nexus 是否适用于大部分国家和地区？

是的，产品设计上面向大部分国家和地区用户，但具体可用性仍可能受到当地法律政策、App Store 上架范围、网络条件和第三方模型服务可用性的影响。

AI Nexus 需要注册账号吗？

不需要。当前版本不要求你单独注册 AI Nexus 账号。你主要需要自己配置模型服务的端点、API Key 和模型名。

默认保存在设备本地。聊天、统计记录、个性化内容和大部分设置都采用本地优先方式处理。同步是可选功能，不是默认强制上传。

联系支持前，最好准备什么信息？

建议附上设备型号、系统版本、AI Nexus 版本、使用的模型提供方、问题复现步骤、报错文案，以及一张能看清状态的截图。

Privacy Policy

隐私政策

以下隐私政策同时适用于 AI Nexus 的 iOS 和 macOS 版本，以及当前这份支持与隐私说明页面。

1. 适用范围 / Scope

本页面适用于 AI Nexus 应用及与之对应的支持与隐私政策页面。 It applies to the AI Nexus app on iOS and macOS and to this support and privacy page.

如果你通过 AI Nexus 连接第三方模型服务，第三方服务对数据的处理也将受其自身隐私政策和服务条款约束。

2. 我们收集和处理的信息 / Information We Collect and Process

2.1 你主动提供的信息 / Information You Provide

  • 聊天内容、提示词、用户信息和个性化内容
  • 接口地址、模型名称、API Key 以及相关模型参数
  • 你填写的资料信息，例如昵称和邮箱
  • 你主动选择的图片或文件附件

2.2 应用在本地生成或保存的信息 / Data Generated or Stored Locally

  • 聊天会话、消息时间、模型切换记录和会话标题
  • Token 用量、费用估算和时间范围统计等使用记录
  • 主题、语言和偏好设置等设备端配置

2.3 敏感配置 / Sensitive Configuration

已保存的模型配置和 API Key 会使用系统安全存储机制保存。

3. 我们如何使用这些信息 / How We Use Information

  • 提供聊天、模型切换、消息生成和多模型访问能力
  • 保存会话历史、偏好设置和个人化上下文
  • 记录 Token 用量和估算费用，帮助你查看使用趋势
  • 在你主动触发时执行模型发现、接口测试和同步操作

4. 第三方模型服务与网络请求 / Third-Party Providers and Requests

AI Nexus 会将请求发送到你自己配置的模型接口，例如 OpenAI、Anthropic、Google 或其他兼容服务。

  • 发送消息时，聊天内容会被发往你选择的模型服务以生成回复
  • 接口测试和模型发现会向你配置的服务端点发起请求
  • 如果你附加图片，且当前模型支持视觉输入，图片数据可能会随请求一起发送
  • 非图片文件在当前版本通常不会被自动解析并上传给模型

这些第三方服务对数据的处理不受 AI Nexus 单独控制。

5. 数据存储、同步与安全 / Storage, Sync, and Security

5.1 本地存储 / Local Storage

聊天记录、统计记录、个性化内容、资料信息和大部分配置默认保存在你的设备本地。

5.2 安全存储 / Secure Storage

已保存的 API Key 和模型配置会通过系统安全存储能力保存。

5.3 可选同步 / Optional Sync

在支持的设备与功能路径下，如果你主动开启自动备份或手动执行同步，应用会尝试同步聊天、配置、 统计记录、资料信息和共享记忆。

5.4 安全说明 / Security Note

我们会尽量采用合理措施保护信息，但任何电子存储或网络传输都无法保证绝对安全。

6. 年龄与地区说明 / Ages and Regions

AI Nexus 面向全年龄段用户。若未成年人使用，建议在父母或监护人指导下管理模型服务、API Key、 网络访问与付费相关配置。

AI Nexus 设计上面向大部分国家和地区用户，但实际可用性可能受当地法律法规、App Store 覆盖范围、 网络连接条件以及第三方模型服务可用性的影响。

7. 你的控制权 / Your Controls

  • 你可以修改或删除聊天内容
  • 你可以清空 Token 统计记录
  • 你可以删除已保存的 API Key 或模型配置
  • 你可以关闭自动同步，或不使用同步功能
