AI Nexus Privacy Policy

隐私政策

本页面说明 AI Nexus 如何处理聊天内容、模型配置、API Key、附件、统计记录以及可选同步数据， 并说明其同时适用于 iOS 和 macOS、面向全年龄段，以及大部分国家和地区用户。 This page explains how AI Nexus handles your conversations, model settings, API keys, attachments, usage records, and optional sync data.

Quick Summary

简要说明

AI Nexus 不要求你注册账户。当前版本默认不集成广告 SDK，也不依赖第三方统计 SDK。 你在应用中输入的内容会按你选择的功能用途处理，例如本地存储、调用你配置的 AI 接口， 或在支持设备上进行可选同步。本政策适用于 AI Nexus 的 iOS 和 macOS 版本。

AI Nexus does not require a separate user account. The current build does not depend on ad SDKs or third-party analytics SDKs by default. Your data is processed according to the feature you use, such as local storage, requests to the model provider you configure, or optional sync on supported devices.

1. 适用范围 / Scope

本隐私政策适用于 AI Nexus 的 iOS 和 macOS 应用，以及与其对应的支持和隐私页面。 This Privacy Policy applies to the AI Nexus app on iOS and macOS and its related support and privacy page.

如果你通过 AI Nexus 连接第三方模型服务，第三方服务对数据的处理也将受其自身隐私政策和服务条款约束。

2. 我们收集和处理的信息 / Information We Collect and Process

2.1 你主动提供的信息 / Information You Provide

  • 聊天内容、提示词、用户信息和个性化内容
  • 接口地址、模型名称、API Key 以及相关模型参数
  • 你填写的资料信息，例如昵称和邮箱
  • 你主动选择的图片或文件附件

2.2 应用在本地生成或保存的信息 / Data Generated or Stored Locally

  • 聊天会话、消息时间、模型切换记录和会话标题
  • Token 用量、费用估算和时间范围统计等使用记录
  • 主题、语言和偏好设置等设备端配置

2.3 敏感配置 / Sensitive Configuration

已保存的模型配置和 API Key 会使用系统安全存储机制保存。

3. 我们如何使用这些信息 / How We Use Information

  • 提供聊天、模型切换、消息生成和多模型访问能力
  • 保存会话历史、偏好设置和个人化上下文
  • 记录 Token 用量和估算费用，帮助你查看使用趋势
  • 在你主动触发时执行模型发现、接口测试和同步操作

4. 第三方模型服务与网络请求 / Third-Party Providers and Requests

AI Nexus 会将请求发送到你自己配置的模型接口，例如 OpenAI、Anthropic、Google 或其他兼容服务。

  • 发送消息时，聊天内容会被发往你选择的模型服务以生成回复
  • 接口测试和模型发现会向你配置的服务端点发起请求
  • 如果你附加图片，且当前模型支持视觉输入，图片数据可能会随请求一起发送
  • 非图片文件在当前版本通常不会被自动解析并上传给模型

这些第三方服务对数据的处理不受 AI Nexus 单独控制。

5. 数据存储、同步与保留 / Storage, Sync, and Retention

5.1 本地存储 / Local Storage

聊天记录、统计记录、个性化内容、资料信息和大部分配置默认保存在你的设备本地。

5.2 安全存储 / Secure Storage

已保存的 API Key 和模型配置会通过系统安全存储能力保存。

5.3 可选同步 / Optional Sync

在支持的设备与功能路径下，如果你主动开启自动备份或手动执行同步，应用会尝试同步聊天、配置、 统计记录、资料信息和共享记忆。

5.4 数据保留 / Retention

数据会保留到你删除相关聊天、清空统计记录、移除安全凭证，或卸载应用为止。 具体表现也可能受系统安全存储机制影响。

6. 安全措施 / Security

  • 应用使用系统安全存储机制保存敏感模型配置
  • 应用支持系统认证能力，用于解锁安全凭证库等敏感区域
  • 任何电子存储或网络传输都无法保证绝对安全

7. 你的控制权 / Your Controls

  • 你可以修改或删除聊天内容
  • 你可以清空 Token 统计记录
  • 你可以删除已保存的 API Key 或模型配置
  • 你可以关闭自动同步，或不使用 iCloud 同步功能

8. 年龄与地区说明 / Ages and Regions

AI Nexus 面向全年龄段用户。若未成年人使用，建议在父母或监护人指导下管理模型服务、API Key、 网络访问与付费相关配置。

AI Nexus 设计上面向大部分国家和地区用户，但实际可用性可能受当地法律法规、App Store 覆盖范围、 网络连接条件以及第三方模型服务可用性的影响。

9. 政策变更 / Changes to This Policy

我们可能会在功能或数据处理方式发生变化时更新本政策。更新后的版本会发布在本页面，并标注最新更新时间。
